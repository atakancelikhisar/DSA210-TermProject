# Clash Royale Match Performance Analysis (DSA 210)

## Project Overview
This project performs a quantitative analysis of my personal Clash Royale match history. The goal is to identify key factors that influence win probability using various Machine Learning techniques.

## Features
- **Exploratory Data Analysis (EDA):** Visualization of card-specific win rates.
- **Machine Learning Models:** Comparison between Optimized Random Forest and Logistic Regression.
- **Performance Evaluation:** Detailed analysis using Confusion Matrices and accuracy scores.

## Files in this Repository
- `DSA210_ML_Analysis.ipynb`: Main Jupyter Notebook containing all analysis and models.
- `atakan_clash_data.json`: The raw dataset of match logs.
- `requirements.txt`: List of necessary Python libraries.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Open the `.ipynb` file in Jupyter Notebook or VS Code.
3. Run all cells to see the analysis results.
